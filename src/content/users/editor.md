# Editor

Trusted contributor with elevated permissions. Can edit any content but cannot delete or manage users.

## Permissions

- ✅ Create any content
- ✅ Read all content (public + private)
- ✅ Update any content
- ❌ Delete content
- ❌ Manage users
- ❌ System settings
